# Knowledge — base de connaissances culinaire

9 fichiers Markdown, distillés à la main depuis des ouvrages de référence, conçus pour être ingérés
par un agent IA (Gemini, Claude, ChatGPT) dans son champ *Knowledge / Project knowledge*.

Le principe directeur : **transformer le savoir en action immédiate**, pas accumuler de la théorie.

| `02-regles-pratiques.md` | Pratique | Assaisonnement, cuisson, rattrapage — diagnostics et ajustements | Synthèse |
| `03-nosrat-gout.md` | Goût | Sel, gras, acide, chaleur — 30 règles + patterns + erreurs | *Salt, Fat, Acid, Heat* — S. Nosrat |

## Chaîne de priorité

Quand plusieurs couches s'appliquent, l'agent les mobilise dans cet ordre :

```
RAISONNEMENT → GOÛT → STRUCTURE → SCIENCE → TECHNIQUE → PATTERNS → ASSOCIATIONS
```

## Note sur les sources

Ces fichiers sont des **notes de synthèse personnelles et reformulées** (règles, heuristiques, repères),
pas des reproductions des ouvrages cités. Les livres sources restent la propriété de leurs auteurs et
éditeurs ; ils sont créditiés ci-dessus à titre de référence. Aucun texte intégral n'est redistribué ici.

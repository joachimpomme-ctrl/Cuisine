# Associations V1 — Pairings et affinités aromatiques

## Rôle de ce fichier

Ce module répond aux questions d'associations : "avec quoi marier X ?", "qu'est-ce qui va bien avec Y ?"

Sources : *The Flavor Bible* (Page & Dornenburg, 2008) et *The Flavor Thesaurus* (Segnit). Extraction ciblée sur les ingrédients les plus utiles en cuisine du quotidien.

Il sert à :
- proposer des combinaisons d'ingrédients pour le Mode Inspiration ;
- enrichir une recette en suggérant des accents aromatiques cohérents ;
- répondre à "j'ai X et Y, qu'est-ce que je peux faire ?" ;
- compléter le Mode Inventaire quand les ingrédients disponibles semblent incompatibles.

Il ne remplace pas :
- les règles de cuisson et d'assaisonnement (voir Nosrat, McGee) ;
- la construction structure du plat (voir Escoffier) ;
- les patterns créatifs (voir Ottolenghi).

## Comment utiliser ce module

- Les affinités en **MAJUSCULES** sont les plus fortes (essentielles).
- Les affinités en minuscules sont solides (très bon accord).
- Pour construire un plat : prendre un ingrédient central, choisir 2-4 affinités complémentaires.
- Pour une question "qu'est-ce qui va avec X ?" : lire la liste + proposer 1-2 combinaisons concrètes.
- Ne pas lister toutes les affinités — choisir les plus pertinentes selon le contexte.

### Romarin

## ÉPICES

### Cumin
Affinités fortes : **CORIANDRE**, **AIL**, **PIMENT**, **CITRON**  
Solides : agneau, bœuf, carottes, cannelle, cardamome, cayenne, légumineuses, lentilles, oignons, pois chiches, poivrons, poulet, tomates, yaourt  
Note : torréfier les graines à sec avant utilisation pour décupler le parfum.  
Combos : cumin + coriandre + piment + ail (base orientale) / cumin + citron + tahini (houmous, sauce)

### Cannelle
Affinités fortes : **POMMES**, **CHOCOLAT**, **MIEL**, **GINGEMBRE**  
Solides : agneau, café, cardamome, carottes, clous de girofle, cumin, dattes, figues, fruits, girofle, miel, muscade, noix, orange, poires, raisins secs, riz au lait, vanille  
Applications salées : agneau braisé, tagine, sauce tomate du Moyen-Orient, ragoûts.  
Combos : cannelle + cumin + coriandre + gingembre (tagine) / cannelle + chocolat + café (mole)

### Gingembre
Affinités fortes : **AIL**, **SAUCE SOJA**, **SÉSAME**, **CITRON VERT**  
Solides : coriandre, cumin, curcuma, garam masala, herbes asiatiques, lait de coco, lime, miel, muscade, oignons, orange, piment, poivre noir, poulet, saumon, sésame, viandes  
Applications : gingembre frais = finition vive ; gingembre en poudre = chaleur de fond.  
Combos : gingembre + ail + sauce soja + sésame (base asiatique) / gingembre + citron + miel (marinade/sauce)

### Paprika / Paprika fumé
Affinités fortes : **AIL**, **OIGNONS**, **TOMATES**, **PIMENT**  
Solides : agneau, bœuf, beurre, chorizo, crème, cumin, fenouil, fromage, herbes, laurier, légumineuses, origan, poivrons, porc, poulet, vinaigre de xérès  
Note : paprika fumé (pimentón) apporte une profondeur fumée distincte — à utiliser en finition.  
Combos : paprika fumé + ail + tomates (base espagnole) / paprika + cumin + coriandre (ras el hanout simplifié)

### Curcuma
Affinités fortes : **AIL**, **GINGEMBRE**, **CORIANDRE**, **CUMIN**  
Solides : cannelle, cardamome, cayenne, chou-fleur, coco, curry, fenugrec, herbes, lait de coco, lentilles, oignons, pois chiches, poivrons, poulet, riz  
Note : colore puissamment — 1/4 cuillère à café souvent suffisant.

### Safran
Affinités fortes : **CITRON**, **AIL**, **FENOUIL**, **TOMATES**  
Solides : amandes, asperges, beurre, bouillon, crème, herbes, huile d'olive, oignons, orange, poisson, poulet, riz, vin blanc  
Note : activer dans un liquide tiède (eau, bouillon, crème) 10 min avant utilisation.  
Combos : safran + fenouil + tomates + huile d'olive (bouillabaisse) / safran + citron + amandes (riz pilaf)

## AGENTS DE GOÛT

### Ail
Affinités fortes : **HUILE D'OLIVE**, **CITRON**, **HERBES FRAÎCHES**, **TOMATES**  
Note : l'ail s'associe à quasiment tout en cuisine salée. Se concentrer sur le mode d'utilisation :
- ail cru → puissant, piquant (aïoli, tzatziki)
- ail sauté → doux, aromatique (bases de sauce)
- ail rôti entier → fondant, caramélisé, moins piquant (tartinable, sauces)
- ail confit → très doux, onctueux

### Anchois
Affinités fortes : **AIL**, **CÂPRES**, **CITRON**, **TOMATES**  
Solides : basilic, beurre noisette, fenouil, oignons caramélisés, olives, parmesan, persil, poivrons rouges rôtis, romarin, vinaigre  
Note : à utiliser fondu dans l'huile ou le beurre pour un fond umami invisible — les anchois "disparaissent" mais enrichissent le goût.  
Combos : anchois + ail + tomates (sauce puttanesca) / anchois + câpres + citron + persil (gremolata étendue)

### Câpres
Affinités fortes : **CITRON**, **ANCHOIS**, **PERSIL**, **MOUTARDE**  
Solides : ail, citron, crème, estragon, herbes fraîches, huile d'olive, laurier, oignons, olives, thon, tomates, vinaigre  
Combos : câpres + anchois + olives (tapenade) / câpres + citron + beurre (sauce vierge) / câpres + moutarde + cornichons (sauce gribiche)

### Miso
Affinités fortes : **SAUCE SOJA**, **GINGEMBRE**, **AIL**, **SÉSAME**  
Solides : aubergine, beurre, bouillon dashi, champignons, citron, citrus, crème, maïs, miel, oignons, porc, poulet, riz, saumon, vinaigre de riz  
Note : miso blanc (shiro) = plus doux, plus sucré ; miso rouge = plus salé, plus profond.  
Combos : miso + beurre + herbes (umami discret) / miso + gingembre + sauce soja + ail (marinade)

### Sauce soja
Affinités fortes : **AIL**, **GINGEMBRE**, **SÉSAME**, **SUCRE**  
Solides : citron vert, coriandre, herbes asiatiques, miel, mirin, oignons verts, piment, porc, poulet, riz, saumon, vinaigre de riz  
Combos : sauce soja + gingembre + ail + sésame (marinade base) / sauce soja + mirin + sucre (teriyaki)

### Huile d'olive
Affinités fortes : **AIL**, **CITRON**, **HERBES FRAÎCHES**, **TOMATES**  
Note : le véhicule aromatique principal de la cuisine méditerranéenne. Utilisée froide en finition, elle apporte un accent de qualité distinct de son usage en cuisson.

### Vinaigre balsamique
Affinités fortes : **TOMATES**, **FRAISES**, **PARMESAN**, **RÉDUCTIONS**  
Solides : figues, framboises, fromage bleu, herbes, huile d'olive, oignons caramélisés, olives, poivrons rôtis, viandes grillées  
Note : ne jamais bouillir — ajouter en fin de cuisson. Quelques gouttes suffisent souvent.

## ASSOCIATIONS RAPIDES (TRIADES)

Format : ingrédient A + B + C → registre culinaire

- poulet + citron + ail → méditerranéen classique
- agneau + cumin + coriandre → Moyen-Orient / Nord-Afrique
- porc + pommes + moutarde → Normandie
- saumon + aneth + câpres → scandinave
- aubergine + tomates + ail → Méditerranée
- pois chiches + cumin + citron → Levant
- champignons + ail + persil → France / Italie
- courgette + feta + menthe → Grèce
- carottes + gingembre + coriandre → Moyen-Orient
- épinards + parmesan + noix de muscade → Italie classique
- fenouil + orange + olives → Provence / Sicile
- feta + pastèque + menthe → été méditerranéen
- poulet + gingembre + sauce soja → Asie
- bœuf + vin rouge + thym → classique français
- crevettes + ail + piment + lime → Asie du Sud-Est

## Limites

- Ce module couvre principalement les associations de la cuisine occidentale, méditerranéenne et de quelques cuisines asiatiques.
- Les cuisines d'Amérique latine, d'Asie du Sud et d'Afrique sont sous-représentées.
- Les intensités relatives (affinité forte vs faible) se basent sur les indications du Flavor Bible (MAJUSCULES dans l'original) et sur l'expérience culinaire documentée.
- Pour les pairings de vin, le module n'est pas la bonne source.

### PROTÉINES

### Caille
Affinités fortes : **THYM**, **HUILE**, **VINAIGRE (balsamique, vin rouge, xérès)**
Solides : ail, bacon, champignons sauvages, échalotes, estragon, figues, foie gras, grenade, marjolaine, oignons grelots, orange, pancetta/prosciutto, polenta, raisin, sauge, sumac

Combos : caille + bacon + ail + citron / caille + figues + vinaigrette / caille + roquette + grenade

### Poitrine de porc / porc à mijoter
Affinités fortes : ail, sauce soja, anis étoilé, gingembre, pommes
Solides : agrumes, bacon, betteraves, carottes, céleri, cumin, fenouil, laurier, légumes racines, oignons, paprika, pommes de terre, saké, thym, vinaigre
Note : morceau gras = cuisson lente (braisé, confit). Les saveurs vives (agrumes, vinaigre, anis, soja) coupent le gras.
Combos : poitrine + soja + gingembre + anis étoilé / poitrine + pommes + fenouil

### Foie de veau
Affinités fortes : **OIGNONS**, **VINAIGRE** (balsamique, vin rouge, xérès)
Solides : ail, bacon/pancetta, beurre, câpres, cerfeuil, ciboulette, crème, échalotes, figues, citron, persil, pommes, roquette, sauge
Note : cuire très brièvement (~1 min/face), sinon durcit.
Combos : foie de veau + oignons + beurre (à la lyonnaise) / foie de veau + sauge + beurre / foie de veau + figues + oignons + vinaigre

### Foie (volaille, veau, agneau)
Affinités fortes : **OIGNON**, **LARD/BACON**, **POMME**, **AIL**
Solides : betterave, brandy/madère, paprika, persil, polenta, raisin, sauge, thym, yaourt + épices
Note : partenaires robustes pour équilibrer le goût ferreux ; oignon fondu contraste l'amertume. Cuisson COURTE. Trempage 1 h dans du lait adoucit un foie trop fort.
Combos : foie + oignons fondus + polenta (fegato alla veneziana) / foies de volaille + ail + thym + brandy (pâté)

### Boudin noir
Affinités fortes : **POMME**, **OIGNON**, **POMME DE TERRE**, **LARD/BACON**
Solides : chocolat noir, coquilles Saint-Jacques, foie, fèves, légumes-racines doux, œuf, poireau, porc, rhubarbe
Note : saveurs automnales ; sert d'umami pour enrichir un plat de viande. Pomme/rhubarbe acidulées coupent sa richesse.
Combos : boudin + pomme caramélisée + oignon / boudin + poireau + pomme de terre (galettes) / boudin + Saint-Jacques (terre-mer)

### Lapin
Affinités fortes : **AIL**, **MOUTARDE (Dijon)**, **BACON/LARD fumé**, **OIGNONS**, **POIVRE**, **THYM**, **VINAIGRE**, **VIN (rouge/blanc sec)**
Solides : carottes, cidre, échalotes, estragon, haricots blancs, laurier, marjolaine, olives vertes, pancetta, pruneaux, romarin, sauge
Note : viande maigre qui sèche vite — braiser/mijoter, larder. La moutarde est le grand classique français.
Combos : lapin + moutarde + vin rouge / lapin + bacon + romarin / lapin + olives + échalotes + marjolaine

### Gibier (chevreuil, sanglier, lièvre)
Affinités fortes : genièvre, vin rouge, champignons sauvages, ail
Solides : airelles/canneberges séchées, cannelle/quatre-épices, cayenne, cerises, châtaignes, chou rouge, clou de girofle, gin, lentilles, Madère, moutarde Dijon, oignons, persil, poivre noir, sirop d'érable, vinaigre
Note : girofle = richesse, genièvre = épice signature. Un fruit aigre-doux (airelle, cerise) équilibre.

### COMPLÉMENTS AUX ENTRÉES EXISTANTES

### Porc — accents à ajouter
À renforcer : anis/graines de fenouil, poivre blanc, coriandre fraîche.
Note : anis + poivre blanc + oignon cuit font ressortir le côté sucré du porc ; l'acidité (pomme, rhubarbe, tomate) coupe le gras ; la coriandre fraîche tranche dans les plats gras asiatiques.
Combos : porc + ail + sauce soja + vinaigre + laurier (adobo philippin) / porc + graines de fenouil + ail + huile d'olive (rôti italien)

### Agneau — accents à ajouter
À renforcer : abricot sec, aubergine.
Note : anchois planté dans les entailles d'un gigot fond et intensifie la salinité ; les fruits acidulés (abricot sec, griotte) coupent le gras.
Combos : gigot + anchois + ail + romarin (entailles) / agneau + abricots secs + cannelle + amandes (tagine) / agneau + aubergine + tomate + béchamel (moussaka)

### Courgette — accents à ajouter
À ajouter : menthe, aneth, maïs, noix/pécan/pignons (et huile de noix/pécan), safran, Pernod/anis.
Fromages au-delà de parmesan/ricotta : chèvre, gruyère, pecorino, queso fresco, cheddar, feta sèche.
Combos : courgette + pecorino + huile de pécan + pécan / courgette + Pernod + huile de noix

### Câpres — note technique
Câpres salées > câpres au vinaigre (rincer/dessaler 15 min pour révéler le côté herbacé-moutarde). Les ajouter à l'huile FROIDE qu'on monte en température. La cuisson longue intensifie leur goût.

### Garam masala
Composition : cardamome + poivre noir + cannelle + girofle + coriandre + cumin + piments séchés + fenouil + macis + muscade.
Note : ajouter EN FIN de cuisson, jamais en début (la chaleur prolongée tue son parfum).
Usages : base de curry (oignon + ail + gingembre + tomate) ; marinade tandoori (yaourt + poulet).

### COMBOS & TRIADES (ajouts)

Combos salés-sucrés :
- banane + bacon (grillée, banane enroulée de bacon) ; banane + poulet + bacon (chicken Maryland) — le sucré bride la salinité du bacon
- amande + rose = sirop orgeat (base Mai Tai ; version sucrée d'ajo blanco)
- noisette + carotte (carrot cake à l'huile de noisette ; noisettes moulues remplacent 1/4-1/3 de la farine, torréfier d'abord)
- bœuf + poivron vert grillé → brochettes barbecue (poivron en coupelles pour retenir les jus)

Triades régionales :
- Soul food (USA Sud) : socle lardons/ham hocks + oignon + vinaigre de cidre + flocons de piment → verdures braisées + ham hocks ; grits + cheddar + crème + crevettes (shrimp & grits) ; verdures + ham hocks + black-eyed peas
- Cuisine chinoise : socle ail + gingembre + sauce soja + sésame → ail + gingembre + porc (sauté) ; gingembre + vin de riz + sauce soja (marinade) ; soja + sucre (laque) ; anis étoilé + soja foncé + cannelle + sucre roux (red cooking, braisé)
- Base française (méta) : piliers CRÈME, BOUILLON, VIN ; herbes signatures estragon/thym/persil ; aromates échalote/oignon/ail/moutarde ; truffe noire en accent — la qualité du bouillon et du vin porte le plat

Nouvelles triades rapides :
- cerises + amandes + crème → dessert français (clafoutis)
- chocolat + café + caramel → dessert riche
- chocolat + noisettes + orange → praliné
- poire + bleu + noix → salade d'automne
- ricotta + miel + figues → entrée/dessert italien
- châtaignes + pommes + crème → velouté d'hiver
- chipotle + citron vert + mayonnaise → sauce Tex-Mex
- piment + coriandre + citron vert → base latino/asiatique

## COMBOS À AJOUTER AUX FICHES EXISTANTES

### Fromage bleu / Roquefort

## CHOCOLAT & FRUITS À COQUE

### Chocolat noir (dessert et usage salé)
Affinités fortes : **CAFÉ/ESPRESSO**, **CARAMEL**, **CRÈME**, **NOISETTE/AMANDE**, **PIMENT/CHILI**, **CARDAMOME**, framboises (surtout au lait)
Solides : banane, cannelle, cerise, châtaigne, fleur de sel, gingembre, lait, lavande, menthe, noix, noix de coco, orange, poires, vanille, rhum, sucre
Note : équilibrer un dessert intense avec café ou caramel. Une pincée de sel le relève ; piment = profondeur. Usage salé : quelques carrés dans un ragoût de bœuf ou un mole (le gras tempère le piquant) ; cardamome moulue = goût plus "cher".
Combos : chocolat + caramel + café / chocolat + crème + framboises / chocolat + noisettes + orange / chocolat + piment ancho + cannelle (mole/chili) / chocolat + amandes + fleur de sel

### Noix et fruits à coque (principes)
Principe : presque toutes marient sucré ET salé. Toujours torréfier (révèle le goût, évite le ramollissement). Choisir la noix selon la géographie du plat. Lavande OK avec amandes/noisettes/pistaches/noix (pas avec châtaignes).
Usages : salades/soupes (croquant), fromage, desserts crémeux, miel, fruits, café, fromage bleu, gratins, légumes verts, pâtes (pignons), poisson, poulet, raisins secs, roquette, vinaigrette.
Combos : épinards + pignons + raisins secs / salade + chèvre + noix / pâtes + pignons + parmesan

### Noix (en garniture/plateau)
Affinités fortes : **FROMAGE (bleu, chèvre, pâte dure)**, **POIRE**, **POMME**, **CÉLERI**, **MIEL**
Solides : endive/cresson, betterave, raisin, figue, roquette, yaourt/labneh, café
Note : tannin + douceur, coupe le gras des fromages. Torréfier légèrement (développe un goût proche du bleu).
Combos : noix + bleu + endive / noix + poire + fromage / labneh + noix + miel (mezze)

### Noisette
Affinités fortes : **CHOCOLAT**, **CAFÉ**, **POIRE**, **VANILLE**
Solides : abricot, ail (picada), banane, caramel, carotte, cerise, figue, framboise, chèvre, mâche, miel, poisson blanc, romarin, roquette
Note : toujours torréfier (composé clé x10), retirer les peaux amères.
Combos : noisette + chocolat + vanille (praliné) / noisette + poire + chèvre + mâche (salade)

### Noix de pécan
Affinités fortes : **CARAMEL**, **SIROP D'ÉRABLE**, **CHOCOLAT**, **POMMES**
Solides : abricots, bananes, beurre, bourbon, café, cannelle, canneberges, crème, dattes, gingembre, miel, muscade, orange, poires
Note : griller pour décupler. Reine des desserts d'automne (tarte, crumble) et du petit-déjeuner.
Combos : pécan + caramel + érable (tarte) / pécan + pommes + cannelle / pécan + chocolat + bourbon

### Raifort (et wasabi)
Affinités fortes : **BŒUF (rosbif)**, **BETTERAVE**, **POISSON FUMÉ (maquereau, hareng)**, **POMME DE TERRE**
Solides : avocat, bacon, crème aigre, huître, pomme, poisson gras cru, saumon fumé, tomate
Note : cru et en fin de cuisson (perd son piquant à la chaleur). À petites doses, nettoie le terreux de la betterave/pdt et l'iodé du poisson cru.
Combos : raifort + crème aigre (purée plus vive) / raifort + crème aigre + betterave (borscht) / raifort dans un pâté de maquereau fumé

## SAUCES AUX NOIX (techniques)

### Picada (Espagne)
Fruits secs pilés (amandes et/ou noisettes) + ail + pain frit + huile d'olive. Épaissit et parfume ragoûts/mijotés (poulet, poisson), s'incorpore aux boulettes, ou en sauce. Variantes : safran, tomate, persil, pignons.
Type : 15 noisettes + 15 amandes torréfiées + 1 tranche de pain frit + 2 gousses d'ail, piler, ajouter 1 c. à soupe d'huile.

### Aillade / tarator
Aillade (Languedoc) = noix + ail pilés + huile d'olive (et huile de noix). Tarator (Turquie) = + pain trempé, jus de citron, vinaigre de vin rouge, un peu de bouillon avant l'huile. Sert avec viande rôtie, légumes (épinards, poivrons rouges, pois chiches).
Type : 1 tasse de noix + 4 gousses d'ail pilées, assaisonner, ajouter lentement 2/3 tasse d'huile.

### Sauge
Affinités fortes : **PORC**, **BEURRE NOISETTE**, **COURGE/POTIMARRON**, **VEAU/VOLAILLE**

### Romarin (compléments)
À ajouter à l'entrée existante :
- Sucré : chocolat noir, orange, citron — fraîcheur eucalyptus-pin relève crèmes/ganaches.
- Caractère proche de la sauge mais plus sucré et floral.

### Thym (compléments)
À ajouter à l'entrée existante :
- Orange/mandarine : bon avec volaille corsée (pintade, dinde).
- Olive : base méditerranéenne réussie.
- Sucré/laitier : note de pâturage frais (glace, crème, chocolat noir).
- Le thym séché peut surpasser le frais dans les mijotés.

### Fromages — règles de mariage (plateau)
- Le fromage aime fruits frais/secs (raisins, figues, dattes, poires, pommes), noix, miel, confiture, mostarda.
- Affiné/fort = sucré/fruité pour contraster (pecorino, bleu → miel, confiture, vin doux). Plus le bleu est intense, plus le miel doit être léger.
- Miel châtaignier = vieux fromages ; miels doux = fromages jeunes/pâte molle.
- Classique : bleu (Roquefort/Stilton) + pain aux noix + porto.

### Fromages — affinités par famille
- Bleu : **POIRES**, **NOIX** (Stilton), miel châtaignier, figues (Gorgonzola), pommes, céleri, châtaignes, porto, steak. Combo : gorgonzola + menthe + noix.

- Mozzarella : **BASILIC**, **TOMATES**, huile d'olive, poivrons rôtis, charcuterie, olives, origan.
- Parmesan : pâtes, risottos, prosciutto, poires, figues, dattes, balsamique vieux, miel châtaignier, noix, melon, fruits à noyau.
- Gruyère/comté : oignons (soupe), ail, jambon, pommes, soufflés/fondues, noisettes, noix, chocolat noir (affiné), thym.
- Ricotta : miel, citron, fruits secs, pignons, épinards, basilic, café, chocolat noir, prosciutto, Prosecco.
- Cheddar : **POMMES**, bacon, chutney, cidre, oignons, poires, noix, macaroni, cayenne, mostarda.

## DIVERS

### Poivre (règle d'emploi)
- Toujours en fin de cuisson (trop cuit, il masque).
- Blanc : doux, pour poissons blancs, sauces/soupes claires (ne colore pas).
- Noir : complexe et piquant, pour bœuf rôti, thon, viandes rouges, steaks.
- Vert (en grains) : moins vif, viandes grillées, canard, gibier, sauces crémeuses, brandy.
- Une pincée de noir rehausse aussi les fruits frais (cerises, fraises).
- Crée une fausse "chaleur" et stimule l'appétit.

### Pickles express (ratio de base)
- 9 vol. vinaigre + 5 sucre + 1 sel + 1 eau.
- Chauffer pour dissoudre, verser tiède sur légumes émincés (oignon rouge, carotte, concombre, betterave crue, tiges de blette).
- Aromate testé : tiges de blette + betterave crue + anis étoilé.
- Prêt après un court repos ; acidité + croquant pour salades, sandwichs, bols.

## ÉPICES (compléments)

### Cardamome
Affinités fortes : **CANNELLE**, **GINGEMBRE**, **ORANGE/CITRON**, **CAFÉ**
Solides : amandes, banane, chocolat, coriandre, cumin, curcuma, dattes, garam masala, miel, pistaches, poires, pommes, riz, safran, vanille, yaourt
Note : pilier du garam masala et des desserts indiens ; parfume café et boissons. Ajouter tôt en cuisson.
Combos : cardamome + cannelle + gingembre (chai) / cardamome + orange + miel / cardamome + café + lait

### Cayenne (piment moulu)
Affinités : **AIL**, **CUMIN + CORIANDRE**, tomates, fromage et sauces fromagères, basilic, viandes, crustacés/fruits de mer, haricots, maïs, œufs, oignons, poivrons, pommes de terre, riz
Note : une pincée en fin de cuisson « booste » et accélère les autres saveurs (renforce le basilic). Plus il cuit, plus il chauffe → doser petit et tard.
À éviter : caviar, truffes, saveurs délicates (il les masque).
Combo : cayenne + coriandre + cumin + ail (base épicée)

### Piments frais/séchés (chile) — repères
Affinités fortes : **HARICOTS** (noirs, pinto), **AIL**, **GINGEMBRE** (Asie), **CORIANDRE fraîche** (Amérique latine), **LIME**, cumin, chocolat, lait de coco
Combo universel : piment + coriandre + lime
Repères par type : ancho = poblano séché, doux-sucré (mole, chili) ; chipotle = jalapeño fumé séché, fort + fumé (poulet, porc, chocolat, haricots, avocat) ; Espelette = doux, légèrement fumé, en finition (peperonata).
Règle : ajouter en fin de cuisson.

### Carvi (graines de cumin des prés)
Affinités fortes : **CHOU/CHOUCROUTE**, **PAIN DE SEIGLE/pumpernickel**, **POMMES DE TERRE**, **OIGNONS**
Solides : ail, betterave, carottes, coriandre, cumin, fromage (Munster, Liptauer), goulash/paprika, lard/porc, pomme, saucisses, thym, tomates, navets
Note : registre est-européen/alsacien. Ajouter en fin de cuisson. La lavande peut le remplacer en pâtisserie.
Combos : carvi + chou + pomme / carvi + paprika + porc (goulash) / carvi + pommes de terre + oignons

### Vanille
Affinités fortes : **CRÈME/LAIT**, **CHOCOLAT**, **FRUITS** (fraise, rhubarbe, pomme, poire, cerise, abricot, pêche), **CAFÉ**
Solides : amande, banane, caramel, cardamome, coco, figue, framboise, gingembre, miel, muscade, orange, safran
Note : adoucit l'acidité, arrondit (custards, glaces). Fendre la gousse, infuser dans un liquide tiède.
Usage salé : pointe dans une sauce tomate (atténue l'acidité) ; beurre blanc à la vanille sur fruits de mer.
Combos : espresso sur glace vanille (affogato) / vanille + rhubarbe / vanille + chocolat

## BASES & PRÉPARATIONS

### Polenta
Base neutre qui appelle gras, fromage et umami.
Affinité forte : **FROMAGE** (fontina, gorgonzola, gruyère, mozzarella, parmesan, taleggio)
Solides : beurre, champignons (chanterelles, cèpes, shiitakes), romarin, ail, herbes (marjolaine, persil, thym, ciboulette), mascarpone, noix, poivrons rouges, sauce tomate, saucisses, flocons de piment, viandes braisées/gibier, jaunes d'œuf, miel, huile de truffe/noix
Note : crémeuse (juste cuite) ou refroidie puis grillée/poêlée.
Combos : polenta + parmesan + romarin / polenta + gorgonzola + mascarpone + noix / polenta + chanterelles + huile de truffe

### Saumurage (brining) — repère
Tremper viande/volaille dans eau salée (sel + sucre ou miel) avant cuisson → moelleux, jutosité, goût. L'essentiel est le sel qui détend les fibres ; les aromates restent subtils.
Repère maison : blanc de poulet (peau + os), sel kasher + sucre/miel + eau, 30 min ; rincer, sécher, reposer au frais quelques heures avant cuisson.
Cibles : poulet, dinde, porc, gibier à plumes (morceaux qui sèchent). Sauter pour une volaille déjà grasse.

### Apporter du piquant ("heat")
Le piquant stimule l'appétit et **rehausse les autres saveurs (sel, acide)** ; apporte vivacité/brillance.
Sources interchangeables selon l'intensité : cayenne, piments frais/secs, ail, gingembre, raifort, moutarde forte, oignon cru, poivre noir, flocons de piment, wasabi.
Geste : un tour de poivre en cuisson OU une pointe de piment en finition (ex. jalapeño dans des coquillages au gingembre + citronnelle).

### Poulet — combos supplémentaires (le corps a déjà les affinités de base)
- poulet + crème + morilles / poulet + chanterelles + romarin
- poulet + moutarde + thym / poulet + ail + pancetta + sauge + thym
- poulet + figues + miel + thym + vin blanc (sucré-salé)
- poulet + clous de girofle + romarin + yaourt (épicé doux)
- poulet + pommes + endives + noix (salade) / poulet + crème + pamplemousse + poivre rose
- tagine poulet + olives + citron confit + petits pois (Maroc)
- Marinade (24 h sur poulet saumuré) : ail rôti + origan sicilien + piment + beaucoup de citron + thym + huile d'olive

### Pois chiches — affinités à ajouter (le corps a déjà ail/cumin/citron/coriandre)
Solides supplémentaires : cardamome, clous de girofle, couscous, curry, citron confit, prosciutto/jambon Serrano, paprika fumé, raisins secs, tamarin, pommes de terre, courge d'hiver, pomme/cidre
Combos : pois chiches + ail + menthe / pois chiches + ail + huile d'olive + thym
Note : torréfier le cumin avant de l'ajouter.

## SAISONNALITÉ — HIVER

Temps froid → privilégier cuissons longues et réconfortantes : **braiser, rôtir, mijoter, glacer, cuisson lente**.
Produits : agrumes (orange, mandarine, pamplemousse, clémentine), poires, dattes, racines (carotte, panais, navet, topinambour), courges d'hiver, choux (chou, choux de Bruxelles, brocoli), pommes de terre, lentilles, haricots, champignons sauvages.
Viandes/saveurs : bœuf, porc, gibier (venaison), moules/coquillages (homard, moules), romarin, sauge.
Épices chaudes : poivre noir, cayenne, cannelle, clous de girofle, gingembre, muscade, curcuma.

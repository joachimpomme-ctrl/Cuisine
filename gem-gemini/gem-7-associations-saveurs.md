# Associations V1 — Pairings et affinités aromatiques

## Rôle de ce fichier

Ce module répond aux questions d'associations : "avec quoi marier X ?", "qu'est-ce qui va bien avec Y ?"

Sources : *The Flavor Bible* (Page & Dornenburg, 2008) et *The Flavor Thesaurus* (Segnit). Extraction ciblée sur les ingrédients les plus utiles en cuisine du quotidien.

Il sert à :
- proposer des combinaisons d'ingrédients pour le Mode Inspiration ;
- enrichir une recette en suggérant des accents aromatiques cohérents ;
- répondre à "j'ai X et Y, qu'est-ce que je peux faire ?" ;
- compléter le Mode Inventaire quand les ingrédients disponibles semblent incompatibles.

Il ne remplace pas :
- les règles de cuisson et d'assaisonnement (voir Nosrat, McGee) ;
- la construction structure du plat (voir Escoffier) ;
- les patterns créatifs (voir Ottolenghi).

## Comment utiliser ce module

- Les affinités en **MAJUSCULES** sont les plus fortes (essentielles).
- Les affinités en minuscules sont solides (très bon accord).
- Pour construire un plat : prendre un ingrédient central, choisir 2-4 affinités complémentaires.
- Pour une question "qu'est-ce qui va avec X ?" : lire la liste + proposer 1-2 combinaisons concrètes.
- Ne pas lister toutes les affinités — choisir les plus pertinentes selon le contexte.

## AGENTS DE GOÛT

### Ail
Affinités fortes : **HUILE D'OLIVE**, **CITRON**, **HERBES FRAÎCHES**, **TOMATES**  
Note : l'ail s'associe à quasiment tout en cuisine salée. Se concentrer sur le mode d'utilisation :
- ail cru → puissant, piquant (aïoli, tzatziki)
- ail sauté → doux, aromatique (bases de sauce)
- ail rôti entier → fondant, caramélisé, moins piquant (tartinable, sauces)
- ail confit → très doux, onctueux

### Anchois
Affinités fortes : **AIL**, **CÂPRES**, **CITRON**, **TOMATES**  
Solides : basilic, beurre noisette, fenouil, oignons caramélisés, olives, parmesan, persil, poivrons rouges rôtis, romarin, vinaigre  
Note : à utiliser fondu dans l'huile ou le beurre pour un fond umami invisible — les anchois "disparaissent" mais enrichissent le goût.  
Combos : anchois + ail + tomates (sauce puttanesca) / anchois + câpres + citron + persil (gremolata étendue)

### Câpres
Affinités fortes : **CITRON**, **ANCHOIS**, **PERSIL**, **MOUTARDE**  
Solides : ail, citron, crème, estragon, herbes fraîches, huile d'olive, laurier, oignons, olives, thon, tomates, vinaigre  
Combos : câpres + anchois + olives (tapenade) / câpres + citron + beurre (sauce vierge) / câpres + moutarde + cornichons (sauce gribiche)

### Miso
Affinités fortes : **SAUCE SOJA**, **GINGEMBRE**, **AIL**, **SÉSAME**  
Solides : aubergine, beurre, bouillon dashi, champignons, citron, citrus, crème, maïs, miel, oignons, porc, poulet, riz, saumon, vinaigre de riz  
Note : miso blanc (shiro) = plus doux, plus sucré ; miso rouge = plus salé, plus profond.  
Combos : miso + beurre + herbes (umami discret) / miso + gingembre + sauce soja + ail (marinade)

### Sauce soja
Affinités fortes : **AIL**, **GINGEMBRE**, **SÉSAME**, **SUCRE**  
Solides : citron vert, coriandre, herbes asiatiques, miel, mirin, oignons verts, piment, porc, poulet, riz, saumon, vinaigre de riz  
Combos : sauce soja + gingembre + ail + sésame (marinade base) / sauce soja + mirin + sucre (teriyaki)

### Huile d'olive
Affinités fortes : **AIL**, **CITRON**, **HERBES FRAÎCHES**, **TOMATES**  
Note : le véhicule aromatique principal de la cuisine méditerranéenne. Utilisée froide en finition, elle apporte un accent de qualité distinct de son usage en cuisson.

### Vinaigre balsamique
Affinités fortes : **TOMATES**, **FRAISES**, **PARMESAN**, **RÉDUCTIONS**  
Solides : figues, framboises, fromage bleu, herbes, huile d'olive, oignons caramélisés, olives, poivrons rôtis, viandes grillées  
Note : ne jamais bouillir — ajouter en fin de cuisson. Quelques gouttes suffisent souvent.

## ASSOCIATIONS RAPIDES (TRIADES)

Format : ingrédient A + B + C → registre culinaire

- poulet + citron + ail → méditerranéen classique
- agneau + cumin + coriandre → Moyen-Orient / Nord-Afrique
- porc + pommes + moutarde → Normandie
- saumon + aneth + câpres → scandinave
- aubergine + tomates + ail → Méditerranée
- pois chiches + cumin + citron → Levant
- champignons + ail + persil → France / Italie
- courgette + feta + menthe → Grèce
- carottes + gingembre + coriandre → Moyen-Orient
- épinards + parmesan + noix de muscade → Italie classique
- fenouil + orange + olives → Provence / Sicile
- feta + pastèque + menthe → été méditerranéen
- poulet + gingembre + sauce soja → Asie
- bœuf + vin rouge + thym → classique français
- crevettes + ail + piment + lime → Asie du Sud-Est

## Limites

- Ce module couvre principalement les associations de la cuisine occidentale, méditerranéenne et de quelques cuisines asiatiques.
- Les cuisines d'Amérique latine, d'Asie du Sud et d'Afrique sont sous-représentées.
- Les intensités relatives (affinité forte vs faible) se basent sur les indications du Flavor Bible (MAJUSCULES dans l'original) et sur l'expérience culinaire documentée.
- Pour les pairings de vin, le module n'est pas la bonne source.

### Citron Meyer
Affinités fortes : **MIEL**, **CRÈME**, **VANILLE**, **ORANGE**, **SUCRE**
Note : croisement citron/orange, plus doux et parfume (note orangee) qu'un citron classique. Parfois trop sucre — corriger avec un peu de citron normal ou de lime pour relever l'acidite. Excellent en patisserie (tarte, creme).
Combos : citron Meyer + crème + vanille (tarte) / Meyer + miel + fruits rouges

**Laitue Bibb / sucrine (delicate)**

### Laitue tendre (Bibb, sucrine, beurre)
Affinités fortes : **VINAIGRETTE CRÉMEUSE**, **CITRON**, **FINES HERBES** (cerfeuil, ciboulette, estragon, persil), **ÉCHALOTES**
Solides : avocat, basilic, concombre, orange, radis, graines de sesame
Note : laitue delicate, presque cremeuse — lui associer une sauce cremeuse (mayonnaise legere, yaourt) plutot qu'une vinaigrette acide agressive ; radis pour le croquant et le piquant.
Combos : laitue beurre + vinaigrette cremeuse + fines herbes + radis

**Triades cuisine du Sud des USA (Soul food)**

### Registre Sud américain (soul food)
Synthèse de plusieurs fiches (verdures, grits, cuisine du Sud) plutôt qu'une entrée unique.
Base typique : lardons/salt pork/ham hocks, oignon jaune, vinaigre de cidre, flocons de piment.
- Verdures braisées (collard, mustard) + ham hocks/lardons + oignons + vinaigre de cidre
- Grits (semoule de maïs) + cheddar/parmesan + crème + crevettes ("shrimp & grits")
- Verdures + ham hocks + pois (black-eyed)
Note : registre réconfortant, gras fumé + acide + amertume des verdures.

**Fruits secs (raisins secs, abricots, pruneaux)**

### Fruits secs (raisins secs, abricots, pruneaux, dattes)
Affinités (fiche source, minimale) : cannelle, gingembre, noix et pistaches, citron, jus de pomme, chocolat, vanille.
Note : si le fruit est dur, le réhydrater (vapeur, eau tiède) avant emploi. Apporte sucre et moelleux aux tajines, farces, riz pilaf et pâtisseries.
Combos (usages classiques, extrapolation) : fruits secs + cannelle + noix (farce, granola) / abricots secs + agneau + cannelle (tajine) / pruneaux + porc

**Noix & fromage fort**

Combo : **noix + fromage fort affiné** (parmesan, pecorino, gouda vieux). Les noix fraîches (humides) ont une saveur lactée qui rehausse les fromages puissants ; les noix sèches accompagnent particulièrement bien le gouda affiné à la saveur caramel/butterscotch. Classique sur plateau de fromages, dans une salade endives + roquette + bleu, ou émiettées sur des pâtes au parmesan.

**Amande & asperges**

Combo : **amandes effilées torréfiées + asperges**. La qualité beurrée des amandes grillées complète les notes soufrées des asperges. Faire dorer une poignée d'amandes effilées dans une noix de beurre 6-7 min à feu doux, hors feu ajouter 1 c. à café de jus de citron et du sel, verser sur les asperges cuites.

**Amande & poulet (volaille mijotée)**

Combo : **amandes moulues/pilées + poulet** pour épaissir un mijoté (influence mauresque, gallina en pepitoria). Le poulet doré mijote dans xérès fino + bouillon, puis on ajoute une picada d'amandes (avec ail, safran, clous de girofle, persil) qui lie et parfume la sauce. Les jaunes d'œufs durs écrasés peuvent aussi épaissir en fin de cuisson.

**Amande & romarin (apéritif)**

Combo apéritif : **amandes + romarin**. Amandes frites dans l'huile d'olive (saveur extra-torréfiée), salées, parsemées de romarin finement ciselé — note surprenante de bacon fumé. Compter 1/2 lb d'amandes pour 1 c. à soupe de romarin très finement haché. Servir avec un xérès fino frais.

**Amande & pêche / abricot (fruits à noyau)**

Combo : **amande + pêche/abricot** (les noyaux partagent la saveur amande). Classique italien : pêches au four farcies d'amaretti écrasés dans la cavité du noyau. Les feuilles de pêcher (saveur amande-pêche) infusées dans un sirop sucré parfument sorbets et salades de fruits. Pour l'abricot : sablé/financier aux amandes avec abricots pochés.

**Noisette & poire (salade, vinaigrette)**

Combo : **huile de noisette + poire + fromage de chèvre + mâche**. L'huile de noisette (parfum qui s'atténue à la cuisson, donc à utiliser à froid) se marie particulièrement au vinaigre de framboise pour une salade de chèvre, poire et mâche (qui a elle-même une trace de saveur noisette). Aussi : noisettes torréfiées sur légumes vapeur.

**Noisette & frangipane aux fruits**

Combo pâtisserie : **noisette en frangipane** (alternative aux amandes pour garnir tartes aux fruits, excellente avec figues). Battre 115 g de noisettes mondées torréfiées moulues + 110 g de beurre mou + 100 g de sucre + 25 g de farine + 2 œufs + 1/2 c. à café de vanille ; garnir un fond de pâte sucrée précuit, disposer les figues, cuire 25 min à 175°C.

**Combo — Avocat + concombre (soupe froide)**

À ajouter aux triades/combos rapides :
- avocat + concombre + citron → soupe froide express (mixer chair d'avocat + concombre pelé/épépiné + jus de citron, assaisonner) : crémeux mais rafraîchissant.

**Combo — Avocat + chocolat (mousse crue)**

À ajouter aux combos surprenants :
- avocat + cacao + (banane ou datte) → mousse au chocolat sans cuisson : l'avocat apporte la texture soyeuse, le cacao le goût. Astuce sucrée méconnue.

**Combo — Piment + chocolat / cacahuète (mole)**

À ajouter aux combos surprenants :
- piment séché + chocolat → profondeur du mole mexicain.
- piment séché + cacahuète → sauce satay, et mole de cacahuate (sauce fumée piment + cacahuète). Le gras de la cacahuète tempère le piquant.

**Combo — Coriandre + cumin (base indienne/mexicaine)**

À ajouter aux combos :
- coriandre fraîche + cumin torréfié (+ piment) → base des currys, dhals et chutneys indiens, et des ragoûts de haricots/viande mexicains. Réveille un guacamole terne ; excellent sur des œufs brouillés.

**Combo — Pois cassés + romarin (soupe / accompagnement)**

À ajouter aux combos :
- pois cassés (verts) + oignon + romarin + bouillon → soupe rustique ; cuire avec moins d'eau pour un accompagnement d'agneau ou de porc. Les pois secs ont une saveur plus terreuse et carnée que les pois frais ; le romarin (notes boisées/eucalyptus) leur va bien, tout comme le jambon/lard.

**Combo — Pois + poisson blanc**

À ajouter aux combos :
- petits pois + poisson blanc (+ persil/fenouil/estragon) : la douceur du pois souligne le goût du poisson. Le pois aime le salé — accord remarquable avec la morue/cabillaud salé (bacalao).

**Cuisine thaïe — principe d'équilibre**

### Cuisine thaïe — principe d'équilibre
La cuisine thaïe vise l'équilibre **piquant + acide + salé + sucré** dans chaque plat.
Piliers : **PIMENT**, ail, basilic thaï, citronnelle, coriandre, curry, gingembre, lait de coco, lime, menthe, cacahuètes, riz, sauce de poisson, pâte de crevettes, sucre, curcuma  
Levier : ajuster les 4 axes — lime (acide), sauce de poisson (salé), sucre (sucré), piment (piquant) — jusqu'à l'harmonie.  
Combos : piment + coriandre + lait de coco / piment + curry + sauce de poisson / piment + curry + cacahuètes

**Picada (sauce aux noix/amandes pour épaissir)**

Combo de base espagnol : **picada** = fruits secs pilés (amandes et/ou noisettes) + ail + pain frit + huile d'olive. Sert à épaissir et parfumer ragoûts et plats mijotés (poulet, poisson), à incorporer aux boulettes avant façonnage, ou en sauce sur viande/fruits de mer. Variantes avec safran, tomate, persil, pignons.
Recette type : 15 noisettes + 15 amandes (torréfiées, mondées) + 1 tranche de pain frit + 2 gousses d'ail, piler en pâte, ajouter 1 c. à soupe d'huile d'olive et assaisonner.

**Aillade / tarator (sauce noix-ail)**

Combo méditerranéen : **aillade** (Languedoc) = noix + ail pilés + huile d'olive (et huile de noix). Variante turque **tarator** = on ajoute pain trempé, jus de citron, vinaigre de vin rouge et un peu de bouillon avant l'huile. Sert avec viande rôtie, légumes, ou plat de légumes (épinards, poivrons rouges, pois chiches).
Recette type : 1 tasse de noix + 4 gousses d'ail pilées, assaisonner, ajouter lentement 2/3 tasse d'huile d'olive jusqu'à consistance de sauce.

**Poivre : quand l'ajouter et lequel choisir**

### Poivre (regle d'emploi)
- **Toujours ajouter en fin de cuisson** : trop cuit, le poivre peut masquer les saveurs.
- **Poivre blanc** : plus doux ("plus discret"), pour poissons blancs (fletan), sauces/soupes claires (ne colore pas, ne domine pas).
- **Poivre noir** : plus complexe et piquant, pour boeuf roti, thon, viandes rouges, steaks grilles.
- **Poivre vert** (en grains) : moins vif que le noir, va avec viandes grillees/rouges, canard, gibier, sauces cremeuses, brandy.
- Une **toute petite pincee de poivre noir** rehausse aussi les fruits frais (cerises, fraises) et certains desserts.
- Le poivre cree une fausse "chaleur" et **stimule l'appetit**.

**Ratio de saumure rapide pour pickles maison**

### Pickles express (ratio de base)
Proportion polyvalente (inspiree de l'assaisonnement du riz a sushi japonais) pour mariner rapidement des legumes crus :
- **9 volumes de vinaigre + 5 de sucre + 1 de sel + 1 d'eau**
- Faire chauffer pour dissoudre, verser tiede sur les legumes eminces (oignon rouge, carotte, concombre, betterave crue, tiges de blette...).
- Aromate teste par le chef : tiges de blette + betterave crue + anis etoile.
- Pret apres un court repos ; apporte acidite et croquant a salades, sandwichs, bols.

**Beurre noisette (beurre brun)**

- Combos salés : vinaigrette chaude minute (brunir le beurre, ajouter balsamique + sel + poivre, émulsionner dans la poêle — excellente sur flétan, Saint-Jacques, crabe mou) / beurre noisette + chapelure + persil (sur légumes type chou-fleur)

**Fromages — règles de mariage (sucré/contraste)**

Principes de mariage du fromage (plateau / apéritif)
- Règle générale : le fromage aime les fruits frais et secs (raisins, figues, dattes, poires, pommes), les noix (noix, noisettes), le miel, la confiture et la mostarda (fruits à la moutarde).
- Affiné/fort = sucré/fruité pour contraster : un fromage affiné ou puissant (pecorino, bleu) demande quelque chose de sucré (miel, confiture, vin doux type Passito). « With aged or strong cheese, you need something fruity or sweet to contrast. »
- Plus le bleu est intense, plus le miel doit être léger et délicat.
- Le miel châtaignier va avec les vieux fromages, pas avec un jeune ; les miels jeunes/doux vont mieux avec les fromages jeunes et à pâte molle.
- Jeune/frais = pain simple, miel léger.
- Combo classique : bleu (Roquefort/Stilton) + pain aux noix + porto.

**Saumurage (brining) — viandes maigres**

Saumurage (brining) — repère
- Quoi : tremper viande/volaille dans de l'eau salée (sel + sucre ou miel) avant cuisson pour augmenter moelleux, jutosité et goût.
- L'essentiel est le sel qui pénètre et détend les fibres protéiques ; les aromates restent secondaires et subtils.
- Repère maison : un blanc de poulet avec peau et os, sel kasher + sucre/miel + eau, 30 min ; rincer, sécher, laisser reposer au frais quelques heures avant de cuire.
- Cibles utiles : poulet, dinde, porc, volailles, gibier à plumes — surtout les morceaux qui tendent à sécher.
- Quand sauter : une volaille déjà grasse et naturellement juteuse n'en a pas besoin.

Note : base aromatique (mirepoix) ; cru en salade il apporte croquant et fraîcheur.

### Céleri-rave
Affinités fortes : **MOUTARDE DIJON**, **CITRON**, **POMMES DE TERRE**, **MAYONNAISE/CRÈME**

Note : la base du céleri rémoulade (râpé cru + mayo + moutarde + citron). En purée avec les pommes de terre.
Combos : céleri-rave + citron + mayo + moutarde (rémoulade) / céleri-rave + pommes de terre + crème / céleri-rave + pomme + noix

**Carvi (graines)**

### Carvi (graines de cumin des prés)
Affinités fortes : **CHOU/CHOUCROUTE**, **PAIN DE SEIGLE**, **POMMES DE TERRE**, **OIGNONS**
Solides : ail, betterave, carottes, coriandre, cumin, fromage (Munster), goulash, lard/porc, paprika, pomme, saucisses, thym, tomates
Note : registre est-européen / alsacien. Ajouter en fin de cuisson. Très bon avec le chou et les charcuteries.
Combos : carvi + chou + pomme / carvi + paprika + porc (goulash) / carvi + pommes de terre + oignons

**Mûres**

### Mûres
Affinités fortes : **CITRON (jus)**, **SUCRE (brun, blanc)**, **CRÈME/GLACE**, **POMME/PÊCHE**
Solides : amandes, cannelle, chocolat (noir, blanc), fromage de chèvre/frais, gingembre, mascarpone, menthe, vanille, yaourt
Note : meilleures cuites que crues (crumble, compote, coulis).
Combos : mûres + gingembre + pêches / mûres + miel + vanille + yaourt / mûres + pomme (crumble)

**Gruyère / Comté**

### Gruyère / Comté (montagne)
Affinités fortes : **OIGNONS**, **AIL**, **JAMBON**, **POMMES DE TERRE**
Solides : chocolat noir (gruyère affiné), épinards, fondue, noisettes, pommes, soufflé, soupe à l'oignon, thym
Note : fromage à fondre par excellence (gratins, soufflés, croque, soupe à l'oignon). S'accorde aux confitures de fruits cuits (figue).
Combos : gruyère + oignons + bouillon (soupe gratinée) / gruyère + pommes de terre (gratin) / Comté + jambon + pain (croque)

**Raisin (frais)**

### Raisin (frais)
Affinités fortes : **FROMAGE** (bleu, pâte dure), **NOIX**, **VOLAILLE**, **PORC**
Solides : amande, avocat, fromage frais, pineau/vin, romarin
Note : sa douceur sucrée-acidulée fait contrepoint au gras de la viande et du fromage ; il peut remplacer la tomate hors saison (avec mozzarella, avocat).
Combos : raisin + bleu (stilton, roquefort) + noix (plateau, salade) / volaille ou porc + raisins fondus à la poêle / raisin + noix fraîche (automne)

**Chocolat blanc**

### Chocolat blanc
Affinités fortes : **FRUITS ROUGES** (framboise, fraise), **AMANDE**, **CITRON**, **CARDAMOME**
Solides : café (en contraste), coco, fruits tropicaux (ananas, fruit de la passion), safran, vanille (saveur dominante naturelle)
Note : très sucré (pas de cacao amer) — l'équilibrer avec des saveurs acides/amères ou des fruits sûrs. Les amandes torréfiées cassent le sucre.
Combos : chocolat blanc + framboise / chocolat blanc + amandes torréfiées (+ cerises séchées) → bark / chocolat blanc + citron (gâteau, curd)

**Poire**

### Poire
Affinités fortes : **FROMAGE BLEU**, **CHOCOLAT**, **POMME**, **NOIX**
Solides : amande, cannelle, cidre, coing, miel, vanille
Note : fruit à pépins comme la pomme (mêmes familles d'accords). Le coing (parfum pomme-poire-rose-miel) intensifie une tarte aux pommes ou aux poires.
Combos : poire + bleu + noix (salade, plateau) / poire pochée + chocolat / poire + pomme + coing (tarte)

**Sources de piquant (heat) interchangeables**

### Apporter du piquant ("heat")
Le piquant stimule l'appetit et **rehausse les autres saveurs (sale, acide)** ; il apporte de la vivacite/brillance au plat.
Sources interchangeables selon l'intensite voulue : cayenne, piments frais/secs, ail, gingembre, raifort, moutarde forte, oignon cru, poivre noir, flocons de piment, wasabi.
Geste : un tour de poivre pendant la cuisson OU une pointe de piment en finition (ex. jalapeno dans des coquillages au gingembre + citronnelle) -> ajoute de la brillance sans forcement bruler.

**Pistaches : usages et pieges**

### Pistaches
Affinite forte : **PECHES**
Solides (sucre) : dattes, figues (seches/fraiches), framboises, miel, chocolat (noir/blanc), cardamome, eau de rose, mangues, abricots, oranges, raisins golden
Solides (sale) : anchois, artichauts, asperges, betteraves, roquette, fromage de chevre/parmesan/ricotta/taleggio, poulet, canard, agneau (charcuterie/saucisses), pates, persil
Note : saveur douce -> les utiliser en quantite ou en vedette pour qu'elles ne se perdent pas ; apportent surtout couleur et croquant.
A EVITER : fraises, que la pistache ecrase facilement (preferer framboises).

**Pecan / noix de pecan (versant sale)**

### Noix de pecan (usages sales)
Au-dela des desserts, la pecan torrefiee enrichit des plats sales.
Solides sales : poulet, porc, fromage de chevre, courge butternut, riz sauvage, farces (stuffing), salades, plats sautes (stir-fried), jambon/shiitakes (cf. plat de chef)
Sucre (rappel) : CARAMEL, SIROP D'ERABLE et SUCRE ROUX (affinites fortes), chocolat, banane, pomme, cannelle
Note : la torrefaction reveille leur gout ; pincee de sel pour equilibrer le gras.
Usage sale teste par un chef : pecans + coquilles St-Jacques + crevettes + jambon + shiitakes + echalotes + riz basmati.

**Blettes (bette à carde)**

Blettes / bette à carde (amer)
- Affinités fortes : **AIL**, huile d'olive, citron, parmesan (aussi fontina, gruyère)
- Solides : anchois, bacon, chapelure, cumin, piment, raisins secs, pignons, échalotes, oignons, thym, tomates, vinaigre (balsamique, vin rouge), zeste d'orange
- Combos : blettes + piment + jus de citron / blettes + ail + pignons + raisins secs (style sicilien) / blettes + poivrons + aubergine + pecorino

**Graines de carvi**

Graines de carvi (douces-acidulées)
- Affinités fortes : **PAINS** (seigle, pumpernickel), chou, choucroute, fromage (Muenster, Liptauer)
- Solides : pommes, carottes, coriandre, cumin, oignons, porc, pommes de terre, saucisses, soupes/ragoûts (goulash), thym, tomates, navets, betterave
- Astuce : ajouter en fin de cuisson pour préserver l'arôme.
- Note : la lavande peut remplacer le carvi en pâtisserie.

**Pois chiches — enrichir (épices et combos)**

Pois chiches — à ajouter aux affinités existantes du knowledge
- Solides supplémentaires : cardamome, clous de girofle, couscous, curry/curry en poudre, citron confit, prosciutto/jambon Serrano, paprika fumé, raisins secs, tamarin, pommes de terre, courge d'hiver, pomme/cidre
- Combos : pois chiches + cayenne + ail + citron + huile d'olive + sel + tahini (houmous) / pois chiches + ail + menthe / pois chiches + coriandre fraîche + cumin / pois chiches + ail + huile d'olive + persil / pois chiches + ail + citron + huile d'olive + thym
- Note : torréfier le cumin avant de l'ajouter aux pois chiches.

**Brocoli rabe (rapini)**

### Brocoli rabe / Rapini
Affinités fortes : **AIL**, **HUILE D'OLIVE**, **PIMENT (flocons)**
Solides : anchois, citron, haricots blancs, origan, parmesan, pâtes (orecchiette), pois chiches, saucisse, vinaigre balsamique
Note : amer — l'ail, le piment et l'huile équilibrent. Blanchir puis sauter.
Combos : rabe + ail + piment + huile d'olive / rabe + ail + origan / orecchiette + rabe + saucisse

**Chou (vert / blanc)**

### Chou (vert / blanc)
Affinités fortes : **OIGNONS (rouge)**, **CARVI**, **VINAIGRE (cidre)**, **BEURRE/LARD (bacon)**
Solides : ail, carottes, cheddar, crème, cumin, fenouil, gingembre, lard, moutarde Dijon, paprika, persil, pomme, sauce soja, thym
Note : émincer finement et rôtir à la poêle pour caraméliser les bords (très bon). Carvi + pomme + cidre = registre alsacien/européen.
Combos : chou + lard + oignons / chou + carvi + pomme + cidre / chou + gingembre + sauce soja (sauté asiatique)

**Chou rouge**

### Chou rouge
Affinités fortes : **POMMES**, **VINAIGRE (cidre/balsamique)**, **OIGNONS**, **LARD (bacon/pancetta)**
Solides : ail, baies de genièvre, cannelle/clou, châtaignes, cumin, fromage bleu/chèvre, laurier, miel, sucre brun, noix, vin rouge
Note : braisage classique aigre-doux — l'acide fixe la couleur et équilibre le sucre.
Combos : chou rouge + pommes + vinaigre de cidre / chou rouge + balsamique + sucre brun / chou rouge + lard + bleu + noix

**Cardamome**

### Cardamome
Affinités fortes : **CANNELLE**, **GINGEMBRE**, **ORANGE/CITRON**, **CAFÉ**
Solides : amandes, banane, chocolat, coriandre, cumin, curcuma, dattes, garam masala, miel, pistaches, poires, pommes, riz, safran, vanille, yaourt
Note : pilier du garam masala et des desserts indiens ; parfume aussi café et boissons chaudes. Ajouter tôt en cuisson.
Combos : cardamome + cannelle + gingembre (chai/épices) / cardamome + orange + miel / cardamome + café + lait (boisson)

**Cayenne / piment de Cayenne**

### Cayenne (piment moulu)
Affinités fortes : **AIL**, **CUMIN + CORIANDRE**, **TOMATES**, **FROMAGE**
Solides : basilic, bœuf/viandes, crustacés, fruits de mer, haricots, maïs, oeufs, oignons, poivrons, pommes de terre
Note : une pincée en fin de cuisson 'booste' et accélère les saveurs (pas pour assaisonner lourdement). Plus il cuit, plus il chauffe.
À éviter : caviar, truffes et saveurs délicates.
Combos : cayenne + coriandre + cumin + ail (base épicée) / pincée de cayenne sur fromage/oeufs

**Myrtilles**

### Myrtilles
Affinités fortes : **CITRON (jus + zeste)**, **CANNELLE**, **SIROP D'ÉRABLE**, **MASCARPONE**
Solides : amandes, banane, crème/glace, fromage frais, gingembre, mangue, menthe, miel, oranges, pêches, sucre brun, vanille, yaourt
Note : le citron 'tranche' leur richesse pectineuse ; la cannelle intensifie leur goût.
Combos : myrtilles + cannelle + crème + sucre / myrtilles + citron + sirop d'érable / myrtilles + mascarpone + pêches

**Fromage bleu**

### Fromage bleu (Roquefort, Gorgonzola, Stilton)
Affinités fortes : **POIRES**, **NOIX**, **MIEL**, **ENDIVE/SALADE**
Solides : céleri, châtaignes, crème, dattes, figues, moutarde Dijon, noisettes, pommes, port, raisin, steak
Note : un élément sucré/fruité (miel, poire, figue) contraste avec la force du bleu. Sublime fondu en sauce sur viande rouge.
Combos : bleu + poire + noix / Gorgonzola + menthe + noix / bleu + miel + figues

**Fromage de chèvre**

### Fromage de chèvre (frais / chèvre)
Affinités fortes : **MIEL**, **THYM**, **CITRON**, **BETTERAVE / ROQUETTE**
Solides : amandes, basilic, cerises, figues, herbes, noix, olives, oignons, orange, pignons, poires, pommes, romarin, tomates, vinaigre balsamique
Note : l'acidité (citron, vinaigre) tranche le gras ; miel + noix = classique apéritif. Chaud sur salade.
Combos : chèvre + miel + noix (toast) / chèvre + betterave + roquette + noix / chèvre + thym + cerises

**Mozzarella**

### Mozzarella
Affinités fortes : **TOMATES**, **BASILIC**, **HUILE D'OLIVE**, **ORIGAN**
Solides : anchois, ail, charcuterie (jambon cru, salami), olives, pancetta, pesto, pizza, poivrons rôtis, romarin, sauge, tomates séchées, vinaigre balsamique
Note : burrata = même registre, plus crémeuse (excellente avec tomates, pêche, fèves, huile d'olive).
Combos : mozzarella + tomates + basilic + huile d'olive (caprese) / mozzarella + olives + prosciutto / burrata + tomates + sel

**Rhubarbe**

### Rhubarbe
Affinités fortes : **VANILLE**, **FRAISE**, **GINGEMBRE**, **SUCRE** (indispensable)
Solides : amande, anis, beurre, crème, mangue, miel, orange, safran, sirop d'érable
Usage salé : sa vivacité acide tranche le gras des poissons huileux (maquereau) et de l'agneau. Cuire en compote acidulée (échalote, vinaigre, sucre) servie avec le poisson.
Combos : rhubarbe + fraise + vanille (crumble, tarte) / rhubarbe + gingembre / rhubarbe en compote vinaigrée + maquereau grillé

**Cerise**

### Cerise
Affinités fortes : **CHOCOLAT** (noir), **AMANDE**, **VANILLE**, **CANNELLE**
Solides : café, fromage de chèvre frais, kirsch, noisette, noix
Note : la cerise a une saveur fruitée verte avec un noyau au goût d'amande (benzaldéhyde). Cerises acides (griottes/morello) pour le chocolat et la pâtisserie.
Usage salé : très bonne avec l'agneau et les poissons fumés (chair grasse + acidité de la cerise).
Combos : cerise + chocolat noir (forêt-noire, fondant) / cerise + amande (clafoutis) / griottes + chèvre frais + bon pain (pique-nique)

**Fraise**

### Fraise
Affinités fortes : **CRÈME** (et laitages frais), **VINAIGRE BALSAMIQUE**, **MENTHE**, **NOISETTE**
Solides : amande, basilic, chocolat (blanc surtout), citron, concombre, fromage frais (mascarpone, brie jeune), poivre noir, rhubarbe, sucre
Note : la fraise a des notes caramel/épices ; un peu de sucre, de menthe ou de citron amplifie sa saveur. Le balsamique l'intensifie (quelques gouttes).
Combos : fraise + crème + meringue (Eton mess) / fraise + balsamique + poivre noir / fraise + menthe + citron + sucre (macération)

**Ananas**

### Ananas
Affinités fortes : **VANILLE**, **NOIX DE COCO**, **RHUM**, **CARAMEL**
Solides : avocat, bacon, citron vert, coriandre fraîche, fromage à pâte dure, jambon, menthe, orange, piment, porc, sauge
Note : choisir un ananas mûr (parfum sucré à la base). La bromélaïne attendrit la viande mais empêche la gélatine de prendre et fait cailler la crème.
Usage salé : excellent avec le porc (caramélise et attendrit) ; piment + sel sur un ananas trop acide.
Combos : ananas + porc + coriandre + citron vert (tacos al pastor) / ananas + noix de coco + rhum (piña colada) / ananas + vanille + sucre (gâteau renversé)

**Pastèque**

### Pastèque
Affinités fortes : **FETA**, **MENTHE**, **CITRON VERT**, **CONCOMBRE**
Solides : basilic, coriandre fraîche, oignon rouge, piment, tomate
Note : très aqueuse, elle détrempe les autres ingrédients — associer à des saveurs herbacées et acides, assaisonner au dernier moment. Mêmes affinités que le concombre.
Combos : pastèque + feta + menthe + huile d'olive / pastèque + citron vert + sel (+ pincée de piment) / pastèque + tomate + oignon rouge + coriandre (salade/salsa estivale)

**Vanille**

### Vanille
Affinités fortes : **CRÈME/LAIT**, **CHOCOLAT**, **FRUITS** (fraise, rhubarbe, pomme, poire, cerise, abricot, pêche), **CAFÉ**
Solides : amande, banane, café, caramel, cardamome, coco, figue, framboise, gingembre, miel, muscade, orange, safran
Note : la vanille adoucit l'acidité et arrondit (custards, crèmes pâtissières, glaces). Fendre la gousse et infuser dans un liquide tiède.
Usage salé (subtil) : une pointe de vanille dans une sauce tomate atténue l'acidité ; classique avec les fruits de mer (beurre blanc à la vanille).
Combos : espresso versé sur glace vanille (affogato) / vanille + rhubarbe (crumble) / vanille + chocolat

**Maquereau / poissons huileux**

### Maquereau / poissons huileux
Affinités fortes : **ACIDE** (rhubarbe, groseille à maquereau, citron), **MOUTARDE**, **ÉCHALOTE**, **RAIFORT**
Solides : aneth, betterave, câpres, cresson, fenouil, gingembre, persil, vin blanc, vinaigre
Note : ces poissons gras demandent un contrepoint acide marqué pour trancher le gras. Compote acidulée chaude (rhubarbe ou groseille + échalote + vinaigre) servie avec le poisson grillé.
Combos : maquereau grillé + compote de rhubarbe vinaigrée / maquereau + moutarde + raifort / hareng ou maquereau fumé + betterave + raifort

**Choux de Bruxelles**

### Choux de Bruxelles
Affinités fortes : **LARD/PANCETTA**, **BEURRE**, **AIL**, **VINAIGRE (cidre)**
Solides : amandes, châtaignes, crème, échalotes, érable, fromage (bleu, parmesan, chèvre), moutarde Dijon, muscade, noisettes, oignons, paprika, pomme/cidre, thym
Note : rôtir ou poêler pour caraméliser ; un trait d'acide (citron, vinaigre) en fin réveille l'amertume.
Combos : choux + lard + ail + vinaigre de cidre / choux + pancetta + thym / choux + crème + muscade

**Beurre noisette**

### Beurre noisette
Affinités fortes : **POISSON BLANC (cabillaud, raie, flétan)**, **NOIX/FRUITS SECS**, **VINAIGRE BALSAMIQUE**, **CÂPRES**
Solides : banane, chapelure, chou-fleur, fruits riches, noisettes, persil, poires, Saint-Jacques, sauge
Note : sauce minute — faire brunir le beurre jusqu'à l'odeur de noisette, déglacer au balsamique ou citron, sel/poivre. Parfait sur poisson, Saint-Jacques, raviolis.
Combos : beurre noisette + balsamique + poisson / beurre noisette + sauge (pâtes) / beurre noisette + câpres + citron (raie grenobloise)

**Pomme**

### Pomme
Affinités fortes : **PORC**, **CANNELLE**, **FROMAGE À PÂTE DURE** (cheddar, comté), **MÛRE**
Solides : amande, bacon, betterave, boudin noir, chou (rouge braisé), clou de girofle, crème, fenouil, foie, miel, moutarde, noisette, noix, oignon, orange, poire, raifort
Note : variété acidulée (Granny Smith, reinette) pour la cuisson et pour trancher le gras ; variété sucrée crue. Garder la peau pour le contraste de texture.
Combos : porc rôti + compote de pomme + moutarde / pomme + betterave + noix (salade, vinaigrette huile de noix) / pomme + cheddar + pain (ploughman's)

### saisonnalite

**Cuisine d'hiver — ingrédients et techniques de saison**

## SAISONNALITÉ — HIVER

Quand on cuisine en hiver (temps froid), privilégier cuissons longues et réconfortantes : **braiser, rôtir, mijoter, glacer, cuisson lente**.

Produits de saison : agrumes (orange, mandarine, pamplemousse, citron, clémentine), poires, dattes, légumes racines (carotte, panais, navet, topinambour), courges d'hiver, choux (chou, chou de Bruxelles, brocoli), pommes de terre, lentilles, haricots, champignons sauvages.  
Viandes/saveurs : bœuf, porc, gibier (venaison), moules/coquillages (homard, moules), romarin, sauge.  
Épices chaudes (réchauffantes) : poivre noir, cayenne, cannelle, clous de girofle, gingembre, muscade, curcuma — à privilégier par temps froid.

# Profils de saveurs du monde & bases aromatiques

Guide de cuisine domestique pour orienter un plat vers une cuisine donnée, doser l'assaisonnement selon le registre, et maîtriser le piment. Complète les principes universels (sel-gras-acide-chaleur) par les signatures propres à chaque cuisine.

**Principe directeur :** pour donner un goût « du monde » à un plat, on combine trois leviers :
1. le **socle aromatique** (ce qu'on fait revenir/infuser en premier),
2. la **signature de fin** (herbe, acide ou condiment fermenté ajouté en dernier),
3. le **profil d'équilibre** (quels axes de goût cette cuisine pousse).

---

## 1. Bases aromatiques fondatrices par cuisine

Le « socle aromatique » = ce qu'on fait revenir, infuser ou suer **en premier**, et qui signe l'identité d'un plat. Règle générale : **commencer par le socle, finir par la signature de fin.**

### Europe de l'Ouest
- **France :** mirepoix = oignon + carotte + céleri (suer doucement au gras).
- **Italie :** battuto / soffritto = oignon + carotte + céleri + ail + persil.
- **Espagne :** sofrito = oignon + ail + tomate + poivron (longuement confit).
- **Allemagne :** suppengrün = poireau + carotte + céleri-rave.
- *Signature de fin :* beurre/huile d'olive, herbes fraîches, déglaçage acide.

### Cajun / créole (Louisiane)
- **« Holy trinity »** = oignon + céleri + poivron vert.
- Souvent + ail, + roux (farine-gras coloré) selon le plat.

### Mexique
- **Socle :** oignon + ail + piment (+ tomate ou tomatillo).
- Pâtes de base : recado, adobo.
- *Signature de fin :* coriandre fraîche + jus de lime.

### Inde
- **Inde du Nord :** tarka/tadka = ghee + cumin / graines de moutarde / asa foetida, puis oignon-ail-gingembre + tomate ; **garam masala en fin** de cuisson.
- **Inde du Sud :** huile + graines de moutarde + feuilles de curry + asa foetida + dal/lentilles + noix de coco.

### Asie de l'Est
- **Chine :** gingembre + ail + cébette (+ huile de sésame, sauce soja, vin de riz Shaoxing).
- **Japon :** dashi = kombu + katsuobushi (bonite), base umami de fond.
- **Corée :** ail + gingembre + cébette + gochugaru (flocons de piment) + sauce soja + huile de sésame.

### Asie du Sud-Est
- **Thaïlande / Vietnam :** pâte pilée ail + piment + citronnelle + galanga + échalote.
- *Signature de fin :* nuoc-mâm / nam pla + sucre (de palme) + lime + herbes (coriandre, basilic thaï, menthe).

### Maghreb & Levant
- **Socle :** oignon + ail + cumin / coriandre / paprika.
- Condiments-signatures : harissa, ras el hanout, baharat.
- *Signature de fin :* citron confit, sumac, herbes, huile d'olive/tahini.

### Afrique de l'Ouest & Éthiopie
- **Afrique de l'Ouest :** oignon + piment + gingembre + tomate (+ poudre de crevette, cube bouillon, beurre de cacahuète).
- **Éthiopie :** niter kibbeh (beurre clarifié épicé) + berbere + oignon longuement fondu (sans gras au départ, dans certaines préparations).

> **Mémo express :** « Pour orienter un plat vers la cuisine X, démarre par son socle aromatique et termine par sa signature de fin. »

## 2. Formules d'épices composées (dépannage maison)

Compositions **indicatives** pour reconstituer un mélange ou comprendre ce qu'il apporte. Proportions à ajuster au goût (à confirmer selon les recettes régionales, qui varient beaucoup).

| Mélange | Composition de dépannage | Apport / usage |
|---|---|---|
| **Garam masala** (Inde N.) | cumin, coriandre, cardamome, cannelle, clou de girofle, poivre noir, muscade | Chaud, parfumé ; **ajouter en fin de cuisson** |
| **Ras el hanout** (Maghreb) | cumin, coriandre, cannelle, gingembre, curcuma, paprika, poivre, cardamome, muscade | 10-30 épices dans le commerce ; complexe, chaud |
| **Baharat** (Levant) | poivre noir, cumin, coriandre, cannelle, clou, muscade, paprika, cardamome | Polyvalent viandes/ragoûts |
| **Za'atar** (Levant) | thym/origan séché + sumac + sésame grillé + sel | Acidulé-herbacé ; sur pain/huile, légumes |
| **Cinq-épices** (Chine) | anis étoilé, fenouil, cannelle/cassia, clou de girofle, poivre du Sichuan | Sucré-réglissé ; viandes braisées, canard |
| **Berbere** (Éthiopie) | piment fort, paprika, ail, gingembre, fenugrec, coriandre, cardamome, ajowan | Piquant + profond ; ragoûts (wat) |
| **Curry en poudre** (anglo-indien) | curcuma (couleur), coriandre, cumin, fenugrec, piment, gingembre | Pratique mais ≠ cuisine indienne authentique |
| **Herbes de Provence** | thym, romarin, origan, sarriette, marjolaine (± lavande) | Grillades, légumes rôtis, tomates |
| **Dukkah** (Égypte) | noisettes + sésame + coriandre + cumin + sel | Condiment **croquant** à saupoudrer |
| **Quatre-épices** (France) | poivre, muscade, girofle, gingembre ou cannelle | Charcuterie, terrines, pain d'épices |

### Règles d'usage
- **Épices entières torréfiées à sec puis moulues** > poudres du commerce (plus de parfum, moins d'éventé).
- Mélanges « chauds/parfumés » (garam masala, certains baharat) : **en fin de cuisson** pour préserver les arômes volatils.
- Épices « de fond » (cumin, coriandre, curcuma) : **au début**, dans le gras, pour libérer les arômes liposolubles.
- Conserver moulu < 3-6 mois ; entier se garde bien plus longtemps (à confirmer selon stockage).

## 3. Pondération des axes de goût par cuisine

La mécanique universelle de l'équilibre (sel-gras-acide-chaleur) reste vraie, mais **chaque cuisine vise un point d'équilibre différent**. « Équilibrer l'assaisonnement » dépend du registre ciblé.

### Profils d'équilibre (quels axes la cuisine pousse)

- **Thaï / SE asiatique :** vise l'équilibre **simultané** salé (nuoc-mâm) + acide (lime/tamarin) + sucré (sucre de palme) + piquant + aromatique, à **chaque bouchée**. *Diagnostic : « il manque toujours un des 4-5 axes ».*
- **Japon :** **umami dominant** (dashi), sel modéré, sucré discret, acide léger (vinaigre de riz), gras faible.
- **France classique :** **gras + sel + umami** (fonds, beurre) ; acide en contrepoint (déglaçage, vinaigrette) ; sucré rare dans le salé.
- **Inde :** **aromatique en couches** (épices torréfiées) + gras (ghee) + acide (yaourt, tamarin, tomate) + piquant variable.
- **Mexique :** **acide vif** (lime, tomatillo) + **piment en couches** (sec/frais/fumé) + terreux/torréfié + herbe fraîche en fin.
- **Levant / Maghreb :** **acide** (citron, sumac, grenade) + aromatique + gras (huile/tahini) + touches sucré-salé (fruits secs).
- **Italie :** **umami** (parmesan, tomate, anchois) + huile + acide modéré ; philosophie « moins d'ingrédients, mieux choisis ».

### Heuristique de diagnostic
> Un plat **fade dans un registre donné** manque souvent de l'**axe signature de ce registre**, pas seulement de sel.
> - Plat thaï plat → vérifier les 4-5 axes (souvent acide ou nuoc-mâm).
> - Plat italien plat → manque souvent d'umami ou d'acide.
> - Plat japonais plat → manque souvent d'umami (dashi/soja).
> - Plat mexicain plat → manque souvent d'acide (lime) ou de couche de piment.

## 4. Gestion et étagement du piment / chaleur (capsaïcine)

Le piquant est un **axe d'assaisonnement à part entière**, central dans les cuisines mexicaine, indienne, SE asiatique, sichuanaise et africaine. Il se construit, se dose et se rattrape.

### Étager le piquant
- **Piment de fond :** ajouté tôt, dans le gras (flocons, piment séché, pâte) → chaleur **diffuse** et profonde.
- **Piment de finition :** frais émincé ou huile pimentée en fin → **vivacité et parfum**.
- Construire avec les deux donne de la profondeur sans brutalité.

### Formes et registres (choisir selon la cuisine ciblée)
- **Frais :** vif, végétal (jalapeño, serrano, oiseau).
- **Séché / torréfié :** profond, fumé (ancho, chipotle, guajillo).
- **Fermenté :** gochujang, doubanjiang, sambal — apportent aussi umami et sel.
- **En huile :** chili crisp, huile pimentée — texture + parfum en finition.

### Doser et moduler
- Retirer **graines et placentas (membranes blanches)** réduit nettement le feu.
- **Griller un piment sec** déplace le profil vers le fumé/torréfié.
- **Poivre du Sichuan ≠ piment :** effet anesthésiant/pétillant (*málà* = engourdissant + piquant) ; à doser à part.

### Rattraper un plat trop piquant
La **capsaïcine est liposoluble** : le gras et le sucre apaisent, **l'eau n'apaise pas**.
- Ajouter du **gras** : crème, lait de coco, yaourt, beurre.
- Ajouter un **féculent neutre** : riz, pomme de terre, pain.
- Ajouter du **sucré** et/ou de l'**acide** pour contrebalancer.
- **Augmenter le volume** du plat (plus de base non pimentée).
- **Ne jamais diluer à l'eau** (dilue le goût sans calmer le feu).

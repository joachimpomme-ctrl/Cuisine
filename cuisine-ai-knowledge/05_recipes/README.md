# Recipes & Patterns Knowledge

Cette section contient la base destinée à aider l’agent IA à proposer, adapter et inspirer des plats.

Elle est séparée de `04_rules/` car une recette ou un pattern n’est pas une règle atomique.

## Sous-dossiers

### patterns/

Patterns culinaires génériques réutilisables.

Exemple :
légume rôti + sauce froide + topping croquant

### classics/

Recettes fondatrices fiables.

Exemple :
poulet rôti, omelette, pâtes émulsionnées, légumes rôtis.

### ottolenghi/

Patterns d’inspiration extraits de recettes Ottolenghi.
On extrait la logique culinaire, pas les recettes copiées.

## Principe

- ne pas copier les recettes protégées
- extraire les structures
- privilégier l’adaptabilité
- rendre le contenu exploitable par Gemini

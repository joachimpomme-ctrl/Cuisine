# Matrice des Techniques de Cuisson

## Saisir

- principe : exposer brièvement une surface à une chaleur très vive pour colorer rapidement ;
- objectif culinaire : développer arômes et coloration de surface ;
- variables clés : température de surface, humidité, quantité en poêle, temps avant retournement ;
- erreurs fréquentes : poêle tiède, surcharge, aliment humide ;
- exceptions : préparations sans coloration recherchée ;
- points de sécurité alimentaire éventuels : aucun spécifique hors maîtrise générale de cuisson ;
- règles candidates attendues : chaleur initiale, gestion de l'humidité.

## Rissoler

- principe : cuire dans un corps gras avec coloration progressive ;
- objectif culinaire : dorer régulièrement et construire du goût ;
- variables clés : intensité du feu, taille des morceaux, brassage, quantité de gras ;
- erreurs fréquentes : feu trop bas, morceaux irréguliers, agitation continue ;
- exceptions : bases aromatiques à cuire sans coloration ;
- points de sécurité alimentaire éventuels : prudence si projection de gras ;
- règles candidates attendues : coloration progressive, brassage maîtrisé.

## Revenir

- principe : cuire rapidement au départ dans un peu de matière grasse pour lancer une préparation ;
- objectif culinaire : enrober et faire partir le cru ;
- variables clés : taille des éléments, feu moyen à soutenu, durée courte ;
- erreurs fréquentes : confusion avec suer, cuisson trop longue ;
- exceptions : préparations où l'on veut attendrir sans colorer ;
- points de sécurité alimentaire éventuels : aucun spécifique ;
- règles candidates attendues : rôle de démarrage, différence avec suer.

## Suer

- principe : cuire doucement sans coloration marquée ;
- objectif culinaire : attendrir et faire ressortir le parfum ;
- variables clés : feu doux, sel, humidité propre des végétaux, couvercle éventuel ;
- erreurs fréquentes : brunissement, feu trop fort ;
- exceptions : préparations où une coloration est recherchée ;
- points de sécurité alimentaire éventuels : aucun spécifique ;
- règles candidates attendues : feu doux, arrêt avant coloration.

## Mijoter

- principe : maintenir un léger frémissement en milieu humide ;
- objectif culinaire : attendrir et fusionner les saveurs ;
- variables clés : intensité du frémissement, durée, couvercle, volume de liquide ;
- erreurs fréquentes : forte ébullition, réduction trop rapide ;
- exceptions : réduction finale volontaire plus active ;
- points de sécurité alimentaire éventuels : remise en température complète avant service différé ;
- règles candidates attendues : frémissement doux, gestion du liquide.

## Braiser

- principe : combiner coloration initiale et cuisson lente partiellement immergée ;
- objectif culinaire : attendrir des morceaux fermes et enrichir le jus ;
- variables clés : saisie initiale, quantité de liquide, couvercle, durée ;
- erreurs fréquentes : trop de liquide, pas assez de temps, absence de coloration initiale ;
- exceptions : pièces déjà tendres ;
- points de sécurité alimentaire éventuels : cuisson complète de morceaux épais ;
- règles candidates attendues : rôle de la coloration initiale, volume de liquide.

## Rotir

- principe : cuire par chaleur sèche au four ;
- objectif culinaire : colorer l'extérieur et cuire progressivement l'intérieur ;
- variables clés : température du four, taille de la pièce, position dans le four ;
- erreurs fréquentes : four insuffisamment chaud, ouverture trop fréquente ;
- exceptions : rôtissage volontairement doux de grosses pièces ;
- points de sécurité alimentaire éventuels : cuisson suffisante au coeur selon produit ;
- règles candidates attendues : préchauffage, maîtrise de la coloration.

## Griller

- principe : cuire rapidement par contact ou rayonnement très chaud ;
- objectif culinaire : obtenir marque et goût grillé ;
- variables clés : intensité de chaleur, épaisseur, humidité de surface, durée ;
- erreurs fréquentes : chaleur trop faible, humidité excessive ;
- exceptions : aliments trop fragiles ;
- points de sécurité alimentaire éventuels : vigilance sur pièces épaisses peu cuites ;
- règles candidates attendues : surface sèche, gestion des pièces épaisses.

## Pocher

- principe : cuire dans un liquide chaud sans agitation violente ;
- objectif culinaire : préserver tendreté et tenue ;
- variables clés : température du liquide, acidité éventuelle, durée, fragilité du produit ;
- erreurs fréquentes : ébullition trop forte, manipulation excessive ;
- exceptions : pochage volontairement plus ferme pour aliments robustes ;
- points de sécurité alimentaire éventuels : cuisson suffisante pour oeufs, poisson ou volaille selon contexte ;
- règles candidates attendues : liquide non bouillonnant, préservation des aliments fragiles.

## Bouillir

- principe : cuire dans un liquide en ébullition franche ;
- objectif culinaire : cuire vite ou extraire selon le cas ;
- variables clés : volume d'eau, intensité d'ébullition, salage, taille des morceaux ;
- erreurs fréquentes : confondre avec mijoter, manquer de liquide ;
- exceptions : aliments fragiles préférant frémissement ;
- points de sécurité alimentaire éventuels : cuisson suffisante selon produit ;
- règles candidates attendues : ébullition franche, distinction avec cuisson douce.

## Vapeur

- principe : cuire par chaleur humide sans contact direct avec l'eau ;
- objectif culinaire : préserver intégrité et texture ;
- variables clés : débit de vapeur, taille des morceaux, densité du panier, durée ;
- erreurs fréquentes : panier surchargé, eau touchant les aliments ;
- exceptions : aliments nécessitant coloration ;
- points de sécurité alimentaire éventuels : cuisson uniforme de morceaux épais ;
- règles candidates attendues : hors de l'eau, régularité de taille.

## Four

- principe : environnement fermé de chaleur sèche ou mixte ;
- objectif culinaire : cuisson homogène à grande échelle ou sur durée plus longue ;
- variables clés : préchauffage, circulation d'air, niveau de plaque, inertie ;
- erreurs fréquentes : enfourner avant préchauffage, mauvais positionnement ;
- exceptions : départ à froid pour certains usages spécifiques ;
- points de sécurité alimentaire éventuels : homogénéité insuffisante sur plats volumineux ;
- règles candidates attendues : préchauffage, choix de niveau.

## Basse température

- principe : cuire longtemps à chaleur modérée et stable ;
- objectif culinaire : contrôler la progression de cuisson ;
- variables clés : stabilité de la chaleur, durée, épaisseur, sécurité produit ;
- erreurs fréquentes : température trop fluctuante, extrapolation hasardeuse ;
- exceptions : aliments demandant saisie ou croûte ;
- points de sécurité alimentaire éventuels : prudence élevée sur maintien prolongé et produits sensibles ;
- règles candidates attendues : stabilité thermique, prudence de sécurité.

## Friture

- principe : cuire dans un bain de matière grasse très chaude ;
- objectif culinaire : croûte rapide et texture contrastée ;
- variables clés : température du bain, humidité, taille des pièces, quantité plongée ;
- erreurs fréquentes : bain trop froid, surcharge, aliment humide ;
- exceptions : préparations préférant poêlage ou rôtissage ;
- points de sécurité alimentaire éventuels : risque de projections, cuisson superficielle si pièces trop épaisses ;
- règles candidates attendues : température du bain, gestion de l'humidité.

## Rechauffer

- principe : remettre un plat cuit à une température de service sûre et homogène ;
- objectif culinaire : resservir sans dégradation excessive ;
- variables clés : volume, homogénéité, mélange, puissance, temps ;
- erreurs fréquentes : centre tiède, cycles répétés, attente trop longue après réchauffage ;
- exceptions : préparations servies froides ;
- points de sécurité alimentaire éventuels : point critique majeur ;
- règles candidates attendues : réchauffage uniforme, limitation des cycles.

## Garder au chaud

- principe : maintenir temporairement un plat prêt sans le laisser dériver trop longtemps ;
- objectif culinaire : attendre le service sans casser la texture ;
- variables clés : durée, température de maintien, dessèchement, couvercle ;
- erreurs fréquentes : maintien trop long, dessèchement, baisse excessive de température ;
- exceptions : préparations à servir immédiatement ;
- points de sécurité alimentaire éventuels : point critique important ;
- règles candidates attendues : durée limitée, protection contre dessèchement.

## Deglacer

- principe : dissoudre les sucs avec un liquide après coloration ;
- objectif culinaire : récupérer le goût au fond du récipient ;
- variables clés : chaleur résiduelle, type de liquide, quantité, vitesse d'incorporation ;
- erreurs fréquentes : trop de liquide, fond non assez coloré ;
- exceptions : cuissons sans sucs exploitables ;
- points de sécurité alimentaire éventuels : aucun spécifique ;
- règles candidates attendues : récupération des sucs, quantité de liquide.

## Reduire

- principe : concentrer un liquide par évaporation ;
- objectif culinaire : intensifier goût et texture ;
- variables clés : largeur du récipient, intensité du feu, volume de départ, durée ;
- erreurs fréquentes : feu trop fort, réduction excessive ;
- exceptions : sauces déjà liées ou fragiles ;
- points de sécurité alimentaire éventuels : aucun spécifique ;
- règles candidates attendues : évaporation maîtrisée, surveillance de concentration.

## Confire

- principe : transformer lentement un produit à chaleur modérée dans un milieu gras ou concentré ;
- objectif culinaire : texture fondante et goût développé ;
- variables clés : température modérée, durée longue, milieu de cuisson, taille des pièces ;
- erreurs fréquentes : température trop vive, confusion avec friture ou braisage ;
- exceptions : produits fragiles ne supportant pas longue cuisson ;
- points de sécurité alimentaire éventuels : prudence sur conservation des produits confits ;
- règles candidates attendues : chaleur modérée, distinction avec autres cuissons lentes.

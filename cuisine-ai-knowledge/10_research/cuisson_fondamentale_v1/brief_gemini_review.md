# Brief Gemini — Review du lot `cuisson_fondamentale_v1_raw`

## Mission

Auditer le premier lot brut sur les cuissons fondamentales pour signaler les risques de qualité avant normalisation.

## Points à vérifier en priorité

- doublons exacts ou quasi-doublons ;
- confusion entre techniques proches ;
- règles trop vagues ;
- règles insuffisamment actionnables ;
- exceptions manquantes ;
- sécurité alimentaire insuffisamment prudente ;
- sous-couvertures importantes ;
- classifications discutables.

## Focus métier

- frontière entre `saisir`, `rissoler`, `revenir` ;
- frontière entre `suer`, `mijoter`, `pocher`, `bouillir` ;
- robustesse des règles sur `friture`, `rechauffer`, `garder_au_chaud` ;
- manque éventuel de règles sur `four`, `basse_temperature`, `confire`.

## Sortie attendue

Un rapport structuré indiquant :

- les règles solides ;
- les règles à reformuler ;
- les règles à fusionner ou scinder ;
- les angles morts ;
- les risques de sécurité ;
- la décision proposée avant normalisation.

# Patterns de plats

- viande grillée + sauce courte
- viande braisée + fond de cuisson réduit
- poisson poché + fumet repris en sauce
- volaille pochée + velouté / suprême
- gibier + base brune vive
- légumes + finition courte
- œufs + sauce choisie selon température
- plat froid + sauce de tenue
- rôti + jus
- potage clair + garniture discrète
- potage lié + base de goût + liaison
- entremets + appareil + cuisson douce

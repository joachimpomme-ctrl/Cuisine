# Sources de Référence Cuisine

## Objectif

Ce document définit quelles familles de sources doivent alimenter la base `rules`. L'objectif n'est pas d'accumuler des liens, mais de hiérarchiser les types de références capables de produire des règles fiables, stables et transformables en JSONL.

## Hiérarchie de fiabilité

### Niveau 1 — Fiabilité très élevée

- manuels techniques de cuisine de référence centrés sur les gestes, les procédés et les bases ;
- référentiels pédagogiques de formation culinaire structurée, de type CAP cuisine ;
- documents normatifs et guides institutionnels sur l'hygiène et la sécurité alimentaire ;
- synthèses techniques professionnelles utilisées pour enseigner les fondamentaux.

Usage recommandé :

- base prioritaire pour les règles fondamentales ;
- indispensable pour les sujets d'hygiène, conservation et sécurité ;
- bon support pour dégager des principes robustes et des exceptions.

### Niveau 2 — Fiabilité élevée

- supports pédagogiques de formateurs identifiés ;
- documents de pratique professionnelle de cuisine classique ;
- ouvrages sérieux de technique culinaire orientés transmission de méthode plutôt que recette ;
- contenus méthodiques produits par des écoles, centres de formation ou auteurs techniques crédibles.

Usage recommandé :

- enrichissement des règles techniques ;
- recoupement avec les sources de niveau 1 ;
- utile pour détailler erreurs fréquentes, conditions et gestes d'exécution.

### Niveau 3 — Fiabilité moyenne

- bonnes pratiques domestiques structurées, clairement expliquées et cohérentes ;
- vulgarisation culinaire rigoureuse, quand l'auteur explicite le pourquoi ;
- ressources de transmission de gestes pour cuisine familiale.

Usage recommandé :

- adaptation à l'usage domestique ;
- reformulation plus accessible pour agents grand public ;
- jamais seule pour fonder une règle sensible d'hygiène.

### Niveau 4 — Fiabilité faible ou d'appoint

- blogs amateurs sans cadre méthodique ;
- contenus centrés sur l'expérience personnelle ;
- formats très synthétiques sans justification ;
- conseils d'usage non recoupés.

Usage recommandé :

- source d'hypothèses uniquement ;
- ne pas transformer directement en règle finale ;
- exiger recoupement systématique avec des niveaux supérieurs.

## Types de sources à exploiter

### Livres de cuisine de référence

- techniques de base ;
- mécanismes de cuisson ;
- gestes reproductibles ;
- vocabulaire précis.

### Formations culinaires

- découpage pédagogique clair ;
- hiérarchie des compétences ;
- distinction entre principe, geste, erreur et résultat attendu.

### Pratiques professionnelles

- logique de cuisine classique ;
- enchaînement des gestes ;
- standardisation des procédés ;
- attention portée aux conditions réelles d'exécution.

### Normes hygiène alimentaire

- règles non négociables ;
- limites de sécurité ;
- prudence sur conservation, refroidissement et réchauffage ;
- arbitrage en cas de conflit avec une pratique culinaire séduisante mais risquée.

### Bonnes pratiques domestiques fiables

- transposition des principes à une cuisine familiale ;
- simplification sans déformation ;
- applicabilité avec équipement non professionnel.

## Ce qu'une bonne source doit permettre d'extraire

- un principe culinaire stable ;
- des conditions d'application ;
- une action concrète ;
- des erreurs fréquentes ;
- des exceptions ;
- un niveau de fiabilité proportionné.

## Règle de travail

- ne jamais transformer directement une source en bloc de règles ;
- extraire d'abord les principes ;
- recouper avant de finaliser ;
- traiter l'hygiène et la sécurité comme un domaine à validation renforcée.

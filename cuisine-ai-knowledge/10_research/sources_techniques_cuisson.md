# Sources Techniques de Cuisson

## Objectif

Structurer la recherche sur les techniques fondamentales avant production de règles. Pour chaque technique, l'enjeu est d'identifier un principe stable, un objectif clair et les variables qui changent réellement le résultat.

## Techniques fondamentales

### Saisir

- principe : exposer rapidement la surface d'un aliment à une chaleur vive pour provoquer coloration et concentration aromatique de surface ;
- objectif : développer arômes, couleur et contraste de texture ;
- variables clés : température de surface, humidité de l'aliment, quantité en poêle, matière grasse, temps sans déplacer.

### Rissoler

- principe : cuire dans un corps gras avec chaleur soutenue afin d'obtenir une coloration progressive ;
- objectif : dorer régulièrement et construire du goût ;
- variables clés : intensité du feu, taille des morceaux, quantité en cuisson, brassage, durée totale.

### Suer

- principe : chauffer doucement des aromatiques pour les attendrir sans coloration marquée ;
- objectif : faire ressortir eau et parfum sans goût grillé ;
- variables clés : feu doux, humidité interne, sel, quantité de matière grasse, durée courte à moyenne.

### Mijoter

- principe : maintenir une cuisson douce et régulière dans un milieu humide, avec frémissement léger ;
- objectif : attendrir, fusionner les saveurs, limiter l'agitation ;
- variables clés : intensité du frémissement, durée, volume de liquide, taille des morceaux, couvercle.

### Braiser

- principe : combiner coloration initiale et cuisson lente en milieu partiellement humide ;
- objectif : attendrir des morceaux fermes et enrichir la sauce ;
- variables clés : saisie initiale, quantité de liquide, couverture, durée, température ambiante de cuisson.

### Rôtir

- principe : cuire par chaleur sèche, le plus souvent au four, pour obtenir coloration externe et cuisson interne progressive ;
- objectif : caraméliser l'extérieur tout en maîtrisant le coeur ;
- variables clés : température du four, taille de la pièce, circulation d'air, durée, arrosage éventuel.

### Griller

- principe : cuire rapidement par rayonnement ou contact très chaud ;
- objectif : marquage, goût grillé, cuisson rapide ;
- variables clés : intensité de chaleur, distance à la source, épaisseur, humidité de surface, durée très courte.

### Pocher

- principe : cuire dans un liquide chaud sans ébullition violente ;
- objectif : préserver forme, tendreté et intégrité d'aliments fragiles ;
- variables clés : température du liquide, agitation, acidité éventuelle, durée, taille des pièces.

### Bouillir

- principe : cuire dans un liquide en ébullition franche ;
- objectif : cuire rapidement ou extraire selon le contexte ;
- variables clés : intensité de l'ébullition, volume de liquide, salage, durée, taille des aliments.

### Vapeur

- principe : cuire par chaleur humide sans contact direct avec l'eau ;
- objectif : préserver texture, intégrité et parfois saveur propre ;
- variables clés : débit de vapeur, taille des morceaux, densité de chargement, durée, couvercle.

### Frire

- principe : immerger ou exposer un aliment à une matière grasse très chaude ;
- objectif : croûte rapide et texture contrastée ;
- variables clés : température du bain, humidité de l'aliment, taille des pièces, quantité plongée, durée très surveillée.

### Confire

- principe : cuire longuement à chaleur modérée dans un milieu gras, sucré ou concentré selon le produit ;
- objectif : transformer lentement texture et saveur ;
- variables clés : température modérée, durée longue, milieu de cuisson, teneur en eau, stabilité de chauffe.

### Déglacer

- principe : dissoudre les sucs de cuisson avec un liquide après coloration ;
- objectif : récupérer le goût fixé au fond du récipient ;
- variables clés : chaleur résiduelle, type de liquide, quantité de liquide, vitesse de réduction ensuite.

### Réduire

- principe : faire évaporer une partie de l'eau d'un liquide pour en concentrer goût et texture ;
- objectif : épaissir et intensifier ;
- variables clés : largeur du récipient, intensité de chauffe, volume de départ, agitation, temps.

## Variables transversales à comparer dans les sources

- niveau de chaleur ;
- humidité présente ou évacuée ;
- taille et densité des aliments ;
- quantité chargée dans le récipient ;
- durée réelle de chaque phase ;
- rôle du couvercle ;
- nature du récipient ;
- signe observable d'une bonne exécution.

## Résultat attendu de la recherche

Pour chaque technique, la documentation devra permettre de produire des règles distinctes pour :

- le principe ;
- l'application pratique ;
- les erreurs fréquentes ;
- les exceptions ;
- les usages agentiques pertinents.
